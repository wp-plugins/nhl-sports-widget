The Fantasy Knuckleheads Hockey Sports Widget is an aggregation of the web's best NHL Hockey content.


== Description ==

Imagine the Best NHL based websites from around the Internet aggregated into one place nested into a widget you can easily embed into your website or as a Wordpress Plugin. Well you can stop wishing, Fantasy Knuckleheads has put together an aggregation of the Webs Best Hockey Content ready to easily display on your website.

<b>We have many high quality website nested into our widget, so you're guaranteed to get high quality content from around the web.</b>

The hockey sports widget is an aggregation of our "Hockey Sports Networks" most recent articles.

Special thanks to http://www.nyrnation.net for helping promote this widget and getting things started.


== Installation ==


1. Upload `hockeysports.zip` to the `/wp-content/plugins/` directory via plugin section of your wordpress install
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar

== Frequently Asked Questions ==

= How do I add my website to the Hockey Sports Network? =

Send request to jonathanragus@gmail.com. Only established websites with quality hockey sports content are considered for inclusion. Only those webmasters which run this widget on their website will have their content displayed on the widget.
